# Blog Markdown Guide

Use one `.md` file per blog in this folder.

## Image in blog content

Use standard markdown image syntax with a URL or public path:

`![Your caption](https://example.com/your-image.jpg)`

or

`![Your caption](/img/your-image.png)`

The caption text appears below the image on the blog page.

## Placeholder media card (optional)

If you want the styled preview card with play button, use:

`![Your caption](placeholder:Text inside card)`
